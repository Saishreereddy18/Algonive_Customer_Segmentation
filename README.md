# Algonive_Customer_Segmentation
# Customer Segmentation using K-Means Clustering

## 📌 Project Overview
This project performs customer segmentation using Machine Learning techniques to group customers based on purchasing behavior.

## 📊 Dataset
Mall Customers Dataset containing customer demographics and spending scores.

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## 🤖 Algorithm Used
K-Means Clustering

## 📈 Project Workflow
1. Data Preprocessing
2. Exploratory Data Analysis
3. Optimal Cluster Selection
4. K-Means Clustering
5. Visualization of Customer Groups

## ✅ Result
Customers are divided into different clusters which help businesses target marketing strategies effectively.
